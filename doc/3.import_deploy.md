Para importar e configurar o projeto em outro ambiente (um computador com Windows 10 sem acesso à internet), você pode seguir os passos abaixo:

1. Transfira a pasta de deploy:
   * Copie a pasta deploy para o computador de destino usando um dispositivo de armazenamento externo (como um pendrive ou disco rígido externo).

2. No computador de destino:
   * Navegue até a pasta deploy.
   * Ative o ambiente virtual.
   * Instale as dependências a partir dos pacotes baixados.
   * Execute o seu projeto a partir do ambiente virtual.
   * Aqui está um exemplo de script import_deploy.bat que você pode usar no computador de destino para configurar o ambiente:

``` bash

   @echo off
   REM Define o caminho do arquivo ZIP
   set ZIP_FILE=deploy.zip

   REM Cria uma pasta temporária para extrair o conteúdo do ZIP
   mkdir temp_deploy

   REM Extrai o conteúdo do arquivo ZIP para a pasta temporária
   powershell Expand-Archive -Path %ZIP_FILE% -DestinationPath .\

   REM Navega até a pasta de deploy extraída
   cd temp_deploy\deploy

   REM Ativa o ambiente virtual
   call myenv\Scripts\activate

   REM Instala as dependências a partir dos pacotes baixados
   pip install --no-index --find-links=packages -r requirements.txt

   echo Importação concluída com sucesso!
   pause

```
